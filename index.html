HACKER MANUAL (DEEP WEB HEIST)

Objetivo da Missão

O Hacker é os olhos e ouvidos do Agente de Campo. Seu trabalho é monitorar a rede, desativar sistemas de segurança, quebrar senhas e guiar o Agente pelo prédio corporativo em segurança.

Comandos do Terminal

  • help / manual: Abrem guias de ajuda.
  • netscan: Mapeia a rede para encontrar IPs.
  • brute_force [ip]: Quebra o firewall externo (Senha de 4 dígitos). Requer que o Agente conecte o USB primeiro.
    🟩 = Certo e na posição | 🟨 = Certo, posição errada | 🟥 = Errado.
  • scan_db: Procura no banco de dados o nome da tabela secreta de segurança (Camada 2).
  • sql UPDATE [tabela] SET LASERS='OFF': Injeção de SQL para desativar os lasers do cofre. Substitua [tabela] pela tabela encontrada no scan_db.
  • cat /manuals/door.txt: Lê o manual da porta para saber qual Fio (Vermelho, Verde ou Azul) o agente deve cortar.
  • decrypt_payload: Rouba o arquivo final do cofre (Camada 3). Demora 30 segundos reais e bloqueia o terminal!
  • clear: Limpa a tela.

As 4 Camadas do Assalto

1. Camada 1: Agente interage fisicamente com a Caixa de Rede (USB) -> Hacker faz brute_force (4 dígitos).

2. Camada 2: Hacker acha tabela via scan_db e roda SQL para desligar os lasers vermelhos que bloqueiam os corredores 2D.

3. Camada 3: Hacker lê o manual da porta e grita qual fio físico o Agente deve cortar no mapa 2D.

4. Camada 4 (Final): O Agente chega ao Vault e o defende dos guardas. O Hacker digita align_nodes e resolve o puzzle da rede.

CUIDADO: Vocês têm exatos 15 MINUTOS para fazer tudo isso antes que o timer zere!
